Jokes with Grammar&Expression：

作业讲评：
今天作业讲了很久，因为有太多值得讲的内容了，而且Forrest也最好多看这篇自己写的文章，并且提醒自己不要犯一样的错。需要注意的点如下：
1 -> 拼写！realiy -> really
2 -> 开头结尾句写的内容比较突兀，和上下文都没有逻辑关系，Forrest可以简单点出这篇文章的目的。而且Forrest提到的money在下文也没有说明；
3 -> 短剧太多、口语词汇频率高，比如like、for ages、also、stuff等等；
4 -> 固定搭配容易出错，比如edit a project。课上讲解课在不确定的情况下怎么可以保证不出错；比如可以用“finish a project”；
5 -> 强调了多次学术写作不要写缩写，比如“can’t”、“don’t”等等；
6 -> 区分词性，noisy/noise、efficiently/efficient;
7 -> 否定前缀混淆；能看出Forrest很犹豫是information还是imformation，课上分享了一个判断的小技巧：通过观察嘴唇开合变化决定。比如“in+formation”：in 开、formation开，是一个很流畅的过程；但是inpolite或者imformation就不流畅；
8 -> 内容的逻辑不通顺。比如，看全球通史的时候不明白，应该问老师或者用图书馆电脑查资料，而不是问librarian；另外就是作文开头提到的“money”，也是和全篇文章无关的，除非下文写到用钱置换新设备、买“保持安静”的牌子等等，但“花钱”不是重点；
9 -> 某些句子缺主语；课上引导补充主语或者变成被动语态；
10 -> 结尾没有关于总结的连接词，比如All in all、in conclusion等等；

虚拟语气：

课上补充了虚拟语气的内容，Forrest课后要把表格背下来。在写作中，反证法可以用到虚拟语气。
