作业分析：
*Picture Description
Forrest写图片描述应该比较上手了，这一次的作业依旧完成得不错，有一些细节需要注意：
1. 表达可以更加多样。这次的图片中，有三个人都拿着滑雪板，所以Forrest要描述这个动作三次的话，不妨分别换不同的表达。
2. 注意“背书包”和“穿/戴”的区别，一般背书包不用“wear”。
3. 描写的细节不够。因为图片描述的重点是“描述”，所以不是说“哪里有什么”就可以了，而是要尽量展现出人或物所展现的状态。比如Forrest描写“背书包”，是用“with a backpack”，那书包是处于什么状态呢？被人背着？背在胸前还是背上？还是单手拎着？还是说双手捧在胸前？有很多可能性的呢。

*Article
先表扬Forrest，这次的article作业，完成得不错。文章结构和我们课上讲的能一一对应，逻辑没有很大问题，语法错误也不多。
1. “small classes”是复数，但是Forrest把它当成一个整体了，所以后面接的动词，基本都是用三单了，或者be动词原形，但是这里要用复数才对哦。
2. 在开头段表达不同观点的时候，“xxx, while xxx”是一个表示对比的状语从句，所以是一句话，不要分开变成两个句子呀！另外，Forrest在前半句提到了大班课的好处，后半句想写小班课的好处，但是没提到“小班课”，所以这个对比的逻辑就不完整啦！
3. 在body paragraph 1，分析部分的逻辑顺序，最好换一下。Forrest写的是：
A：班上学生更少的时候，每个人有更多时间回答问题和练习对话。
B：能在课上有更多练习对每个人来说都是很重要的
以上逻辑A和B换过来，会更有说服力，因为：
A → B
具体情况 → 再说重要性
B → A
先重要性 → 再解释小班如何实现
4. 单复数不统一。在第一点“small classes is…”已经有体现了，还有就是“each students”。课上我们都有一一纠正。
5. 虽然Forrest的分析和例子都没有逻辑问题，但是想写四五百字的作文，还是写得不够深入。比如Forrest在例子里面提到“角色扮演”，这在例子里面可以作文一个“总”，然后再具体写小班课如何进行角色扮演，学生怎么通过角色扮演提升语言能力，并且要说明小班课的角色扮演怎么给到学生更多练习的机会。
我们每次课都有一起练习怎么把Forrest的分析和例子扩展一下，Forrest课下也要多对比两个版本，熟悉怎么扩展内容。

期末考试作文练习：
Forrest的期末考试作文和我们课上讲的结构有很大的相似之处。最大的不同点在于，在两段body paragraph中，Forrest要分别论述正反方观点，因为题目的问法和我们经常练习的不一样。
1. 我们先回顾了Forrest在学校学习的写作结构。Forrest能很准确地说出结构的不同，说明他也是熟悉的。但是，Forrest记得，支持的观点要放在body paragraph 2，这样整篇文章的逻辑会更加通顺。另外，我们也强调了不能在学术写作上用感叹号，以及非常口语化的表达，比如“feel like”。
2. 我们还对着Forrest的essay plan分析了Forrest在source中找的quote有没有使用价值，以及source中还有哪些更具参考价值的内容。有个误区是，某一篇文章写到social media对sleep, productivity和mental health的影响，但是Forrest理解成它们之间是有关联的。
确实睡不好会影响mental health，但是如果Forrest要引用某篇文献的话，就要先确定文献有没有相关分析，还是说，以上三者是单独分析的？
另外，Forrest只找出了social media的负面影响的quote，但是第一个body段是写“好处”的，也要找相应的quote。课上我们用不同颜色标注了，方便Forrest写作文的时候，快速定位。
3. Forrest对这篇作文的准备其实挺充分的了，我们课上就可以很快写出开头和结尾。Forrest需要注意的是，开头和结尾提到的内容，是要和两个body paragraph对应的哦！
