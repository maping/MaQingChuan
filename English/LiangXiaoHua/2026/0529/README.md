作业分析：
*Article的任务是重写上次有待改进和补充的BP1～2例子。BP2的结构需要修改，而且两个例子有点类似。
1. 首先还是想指出抄写的错误。虽然对比起上次作业，这方面是有进步的，但是这真的是Forrest可以完全避免的错误！
2. BP1和2的论点虽然不同，但是例子很相似，甚至本质上是一样的！主要是因为，两个论点看似不同，但是关联性很强，Forrest可能自己都觉得难区分。
3 BP2的结构有待改进哦！Forrest经常写完例子，觉得又有新的idea，就来一个“furthermore”，继续写。但是“furthermore”的内容属于“分析”呀！应该在例子前面，而且例子需要体现以上所有内容。
这次的BP2结构就是：论点 - 分析 - 例子 - 分析，显得有点乱。而且“furthermore”的内容，和论点不相关。
BUT！“furthermore”的内容非常棒！可以作为单独的论点来写，而且很好地和BP1的论点区分开来了。BP1是从academic development上分析的，BP2可以从skill development分析。
课上又带着Forrest回忆之前写过的作文，有些也能用到skill development的角度去分析，可以说是一个很有用（或者近似万能）的角度～
比如，我们之前写过，学生是否要每天做运动。Forrest之前就从teamwork/communication skill上分析团体运动的好处和必要性。这篇文章也能从这个角度分析。

课堂作文练习：
Text C - Paddling Alaska
1. 这篇文章的生词有点多呢！不过Forrest上课之前有看过，不影响他对文章的整体理解。这篇文章的难点不在词汇，因为要完成任务的话，有些单词不明白也不影响理解文章，因为它们都不是核心信息；借助上下文语境就能把握内容了。
文章的难点是读懂作者语气。作者好爱开玩笑，玩笑中隐含着讽刺、自嘲、调侃等。另外，时间线也有点难琢磨。课上我们有对文章进行梳理，希望能帮助Forrest理解得更透彻些。
2. Forrest看到这个作者，就联想到张雪峰老师。其实，Forrest还是很关注一些热议的社会事件的。课上Forrest还结合张雪峰老师事件，聊了很多自己对这篇文章/作者的看法。
3. 完成这次Text C任务要注意的是，题目要求写article哦！我们之前讲过journal entry、talk、speech、letter等等，属于比较casual的风格，Forrest也掌握得不错。article会比较formal，还需要Forrest多练练～
