作业分析：
*Article (talent）
这次article作业完成得整体不错，逻辑虽然有点漏洞，但也算蛮清晰的。比较明显的问题是，某些表达不够地道。
1. 表达：文章中不够地道的表达，都一一和Forrest分析了，包括focus strongly on, attend tuition等。有些表达可能受新加坡英语的影响，比如attend tuition，但是，还是希望Forrest按照英式英语或者美式英语的标准来写。
2. 在Body paragraph 1，Forrest的逻辑可以更加缜密。Forrest先分析了training for talent的孩子会怎么样，然后分析了free time 的重要性，接着就是例子。这里如果Forrest补充了没有free time会对孩子造成什么影响的话（造成的后果），会有更加完美的逻辑闭环。

课堂作文练习：
这次课是我们第一次讲阅读中的写作题目。字数要求有点多，但是Forrest能在文章中找到的有用信息其实也很多。
1. 我们先分析了写这封信要注意的内容，包括主角是谁，写给谁，用什么人称，信件的语气如何。Forrest每次拿到作文，都要自己判断写作的语气。有时候同样是letter，写给好朋友和写给校长用的语气不太一样。
2. 接着，我们分析了每一个问题。Forrest要注意，虽然看题目能很清楚地知道我们要回答的问题是什么，但是不意味着Forrest要一个个问题按顺序回答。有时候回答问题一，可能就包含了问题二的答案。比如“看了什么，做了什么”，Forrest在描述自己做的事的时候，也顺带可以把看到的东西描述出来；写感受也是同理。
3. 我们带着问题，一个个在文章里面找答案。Forrest课前看了文章，也做了前面的题，所以对整个故事的主线还是很熟悉的。不过，有些细节，Forrest没注意到。比如Saffron是什么时候开始改变对自己所选专业以及整趟旅程的看法的？Forrest说是看到企鹅的时候。其实比起看到企鹅，更能让Saffron思考的事是，Saffron和John的对话。因为在对话之后，她开始问妈妈问题，而且很专心地听妈妈回答。
4. 整体来说，Forrest对text c的理解比较到位，也能很快在文章里面找到作文中要写的内容的依据，不过要注意的是，不要漏掉细节，还有尽量用自己的话去写。
