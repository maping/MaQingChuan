Free Talk:
今天我们讨论了一下最近网上很火的manifest和北京的offer大楼（中国尊）。然后分析了一下背后的心理学原理。

作业分析：
*Article (Change）
上次的作业写得很好！基本没有需要大篇幅修改的内容，语言表达也很自然。不过有些小问题和论述段落的例子需要改进。
1. Forrest一定要注意标题的大小写。标题是改卷老师最先读到的内容，所以除了方向要对，大小写（细节）也不能出错，这样才能留下比较好的第一印象。课上和Forrest再次复习了标题需要大小写的情况，这些内容要烂熟于心啦！！
2. 表扬Forrest！在body paragraph 1多次提到AI的情况下，首次提到的时候，Forrest写到：artificial intelligence (AI)。做得很棒！在学术写作中，如果要写缩写，必须要先讲到这个缩写的全称是什么，后面才能用缩写。之前我们有讲过，Forrest还记得，很不错哈！
3. Body paragraph 1的例子写得不够充实，而且有点和前面的分析内容混在一起了。Forrest提到的mathematical approaches就是一个很好的例子。可以展开写，说明AI如何解决学生问题，如何根据学生水平调整难易程度和加以引导。Forrest甚至可以对比以前没有AI的时候，学生面临这些问题，是怎么解决的；现在有了AI（change），又是怎样的画面。这样可以突出这个变化对personal growth（开头和结尾提到）的帮助。
4. 对比body paragraph 1，body paragraph 2的例子就好了很多。虽然Forrest有change的前后对比，但是显然变化后的描述不够多。因为Forrest是支持这个变化的，所以建议Forrest多花点笔墨去分析和描述这方面。

*Speech
这一篇演讲稿写得很有画面感，很棒哦！但是也有一些细节需要注意：
1. 时态统一！Forrest有时候不太注意句子前后的时态，比如在描写Charlie以前在海边的经历的时候，要用到过去的时态，但是Forrest句子中，同时出现了动词原形和过去式。说明Forrest不是不会，而是没太注意这些语法细节。
2. 拼写错误。这次拼错的是powerful和fragile。都是Forrest会的单词！希望每次写完都要把自己的文章认真读一遍。
3. Forrest在这篇演讲稿中，用了好几次破折号，因此，我们特意讲了一下什么时候可以用，以及作用是什么。在演讲稿中，用破折号是挺好的，可以表达演讲者的语气，有时候也能强调。但是在article里面，不建议Forrest用哦！因为这样会显得比较casual。
4. 在Text C中，我们已经写过letter和speech了。Forrest对语气和语言风格的整体把握还是挺好的。Forrest能分清这是什么类型的写作，以及是写给谁的。加上有bullet point对Forrest的引导，整体写得都很不错。

课堂作文练习：
这篇Text C和前面四次写得都不一样。这是一种新的类型：magazine article。
1. 这篇文章的难点在于读懂文章。文章表面看着不难，像是在讲故事；但是，很多关键的信息没有直接说出来，而是夹在细节里。所以，Forrest要是对着pullet point的关键词，在文章找“prepare”、“challenge”……基本找不到答案。Forrest需要做的是，如何把文章的描写，转化成题目问的“prepare”、“challenge”等。
2. 虽然题目给了三个bullet point，看似跟着bullet point回答问题就可以了。可是，文章不是按着pullet point的问题顺序来写的。信息都分布在各个角落，有时候一个短论混着几个问题的答案。
3. 课上，我们先讨论了文章大概在讲什么，看看Forrest对文章的整体把握是否准确。虽然文章生词蛮多，但是Forrest预习得很好呀！Forrest对文章写的事情发展把握得很好，而且基本能说出文章的中心思想。
4. 我们带着bullet point找线索的时候，有些埋藏得比较深的内容，Forrest未必能找到。所以，课上老师给了一些关键词引导。比如，哪里能体现出“Lotta’s determination”。
5. Forrest写article的时候，一定要注意，虽然名字是叫article，但是和我们平时练得最多的article类型不太一样。这篇文章是需要结合“article”和“story telling”，去传达一个思想，并带给读者希望：人类对环境的破坏已经到了非常严重的地步，但是我们还有希望～自然会给我们答案，我们要重新学习如何与自然共存。
