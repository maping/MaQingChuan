Free Talk:
今天Forrest放假，我们就聊起了这些节日。Forrest把“believe”和“trust”混淆了，因为中文都是“信”。课上纠正了Forrest，不过也挺喜欢Forrest暴露这种细小的问题的，因为如果Forrest说习惯了，以后就比较难改了。

作业分析：
*Article (Moving to Cities - Rewrite Paragraph 2）
上次课我们讲过Body Paragraph 2的结构之后，Forrest写的第二版有了明显的进步。第二版的主要问题是语言表达。
1. 虽然Forrest重写的时候，调整了结构，但是还是有可以改进的地方。中间关于delivery那句，要么删掉，要么展开写。因为前半部分是关于transportation的，对应“advanced infrastructure”，后半部分hospital/doctor对应“essential services”，所以刚好可以呼应论点。delivery部分虽然也是和论点相关，但是和另外两点比起来有点不足。所以在原本内容比较丰富的情况下，就建议Forrest删掉了。
2. 有些表达还是有中文的思维，我们慢慢改，Forrest也要积累每次讲过的内容。比如：walking 15-minute around distance、complete the trip。
3. 惯用表达也要积累。Forrest很多时候是凭感觉说的，或者Forrest之前一直记的都是错的。比如：是take the subway，from school to the office等。当然，有些冠词是要看语境用， 比如强调“某一趟/某一列地铁”的时候，可以用“a”。这次是针对Forrest这篇文章的语境来讲的。
4. 语义不匹配。Forrest在写“hospital”的时候，把“doctor”和“healthcare”对应起来了。但是，它们的关系其实是：
doctors -> treatment、diagnosis、medical care等
healthcare -> system、policy
所以，写到医生，最好是和病人、诊断这些对应起来。这样写，也可以呼应论点“essential service”。

*Speech
这篇演讲稿虽然修改的内容很多，但是大部分都是建议Forrest修改表达。Forrest在有引导的时候（bullet points），基本上都回答得不错，加上原文能找到依据。
1. Forrest需要花最多精力改进的地方是增加演讲的细节。Forrest想象一下，虽然要回答问题，但是，是以分享自己经历的形式告诉大家。如果只说“overwhelmed”，而不说“dizzy, nausea”等，听众很难感受到challenge具体体现在哪里。
2. 语法和拼写也要注意。不过Forrest最近在这方面的进步很大，很少出现有拼写错误或者很基本的语法错误。这次写错的是被动语态的结构，还有“price”和“pride”。上课的时候，点一下Forrest就能自己更正。

课堂作文练习：
这篇文章对很多内容都藏得很深。Forrest很难再直接通过bullet point的关键词，在文章里找到一模一样或者类似的表达了。
1. 这篇作文的难点在于，Forrest要把原文妈妈在讲的内容，变成自己（Charlie）的回忆和反思，最后达到呼吁的目的。这个转化会有点难。课上我们梳理了文章，Forrest对文章的结构了解得更清楚了。
2. 文章有很多内容不是写得很直接，而是要Forrest自己去理解，特别是关于道理部分。比如从船联系到历史：历史上的船和人；从新的规定联系到现代人对环境的态度。还有些内容是妈妈通过写海边闲逛这件事情，联系到人生。
3. 除了道理是藏在字里行间，文章对心情的描写也是。Forrest找不到“I enjoy …”的字样，但是能找到“exhilarating”、“laughter”、“thrill”等。
