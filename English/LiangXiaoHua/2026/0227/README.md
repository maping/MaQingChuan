作业分析：
*Article (socia media)
先表扬Forrest，这次作业完成得超级快！！！如果能把每次作业都当成考试就好啦！Forrest第一次交这么长的article作业。虽然这次文章内容和结构和我们平时练习的有点不一样，但是整体完成得还不错。以下几个问题，我们在课上一一分析了：
1. Paraphrase：Forrest说，学校老师要求引用参考文献的时候，要paraphrase，而不是照抄原句。其实paraphrase我们练习得也蛮多的啦，在summary里面，经常会有“改写+总结”，而且我们也强调要多用“同义替换”。当然，paraphrase不止是换单词，还可以换句型。课上我们分析了Forrest直接抄参考文献的一些句子，并且分析了怎么paraphrase会更好。Forrest找的参考文献，其实不算特别地学术，比如出现很多口语词汇（say等）。Forrest在引用的时候，是需要按照自己的作文风格改变语言风格的，要尽量和自己整篇文章的风格统一。
2. 逻辑。Forrest找的参考文献都特别长。如果Forrest要在短时间内读明白文章内容和梳理清楚其中的逻辑，很有挑战，课上Forrest也说，这是他第一次读这么长的文章而且是读三篇。所以，Forrest没有搞清楚一些内在逻辑。Forrest还会结合一些自己的常识去分析，但是句子末尾是有引用出处的。比如“睡眠质量差”和“mental health”的关系，就引用错了。
3. Forrest的这篇文章，基本上是自己写论点，然后从参考文献里面“摘抄”一些内容去支撑自己的论点，所以缺乏用自己的话去有条理地论证。这也是为什么我们课上练习了怎么写这篇文章的例子。

课堂作文练习：
*social media作文例子练习：
我们先分析了body paragraph1，找出了“social media”下面的分支：emotional support和creativity。课上我们分别讨论了如何给这两个方面举例子。
Forrest能想起来的例子其实很多，但是需要引导一下。比如老师会先说一个例子，Forrest就能联想到很多内容。
*新作文练习：
今天我们的话题是“talent”。我们主要是分析正反方观点，然后讨论了怎么分析和举什么例子。
1. 我们在分析论点的时候，Forrest很喜欢“例子先行”。每次都是先讲例子。比如这次Forrest举了梅西的例子。例子是很好的，课上引导Forrest要学会怎么倒着分析，将例子浓缩成论点。
2. Forrest写作文的时候，要么就是分析只有两句，要么就是例子很简短。其实分析和例子本质上是同一逻辑内容的不同呈现方式。
