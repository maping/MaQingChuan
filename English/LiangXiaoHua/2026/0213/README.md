作业分析：
*Picture Description
现在Forrest写图片描述基本不会漏什么内容，而且有些很不起眼的东西，Forrest也能注意到，比如这次秋天落叶的图片里面，有一个小小的人，被Forrest注意到了。
1. Forrest这几次的图片描述开头，虽然有用“这是一张xxx图”的句式，但是句子太长咯！如果长句包含了很多信息，Forrest的语法和逻辑容易混乱。所以，Forrest可以写成短句，或者从句，而不是一开始就用有很多伴随成分的长句（比如用with）。
除了开头，Forrest在文章中间也有同样的问题。在句子太长的时候，Forrest就不容易把握清楚各个成分之间的关系。
2. 有些表达不够地道，比如“厚厚的一层落叶”，不用“heavy”，而用“a thick layer of”。

*Article
这一篇article的思路是很不错的，但是从标题开始，就有一个小问题，一直贯穿整篇文章。
1. 题目给的情景没有说到是“数学课”上，所以，它涵盖的应该是所有的“教学”以及包含“师生关系”的事情。如果Forrest在标题就给自己把范围缩小到“数学课”上，读者可能会不禁想到，“数学课才去质疑老师吗？其他课呢？”
另外，Forrest要区分上上一次的“supermarket”那一篇作文，因为题目已经明确说了“超市”，所以标题就可以包含这么具体的内容。
课上，我们对比了这两个题目，分析了它们之间的区别：
*challenging teachers：从大到小 → 用“数学课”例子来体现
*supermarket：从小到大 → 从超市处理临期食品上反映社会问题
2. 因为Forrest在标题就限定了“数学课”，所以后面的point和analysis都是以“数学课”为出发点的。我们在课上分析了如何将本应该作为例子的“数学课”，变成适合更多场景的写法，比如，写成“any subject”。
3. 我们再次强调了，写article不能太绝对。比如“the whole class will get wrong ideas”，这里的“Will”就太绝对了。不排除有同学懂了，但不质疑老师呀。我们提到这一点的时候，Forrest反应蛮快的，马上就改成了“most of the students”。
4. 两个body paragraph的analysis部分都太短了，一句话就结束了，没有分析起因经过结果呀！
5. 在conclusion里面，Forrest犯了一些词性错误。课上我们分析了副词、形容词所修饰的词性。

课堂作文练习：
上一次课，我们练习了“教育”类的写作题目，这次我们继续这个话题，但是题目的形式很不一样，是和“supermarket”类似的，因为提到了“language learning”，有一个具体的场景，而不是任何“learning”。
Forrest在这方面的想法也很多，而且他自己也体验过不同的上课形式了。
1. 课上，我们分析了正反方观点，尽量让Forrest在两方都多想出理由来支持自己的想法。
2. Forrest的想法虽然很多，但是有些是很类似的，不能作为两个单独的论点，放在同一篇文章里；或者，两个论点之间有很密切的联系。
比如，在赞同小班课的时候，Forrest说到小班课上，学生发言的机会多，以及师生之间更容易建立信任。
这两点之间虽然可以单独作为论点，但是Forrest可以将两者的逻辑关系联系起来：因为人少，师生之间的信任更容易被建立起来，所以学生更爱开口练习口语，而不会害羞和怕犯错。
所以，课上老师是很喜欢听Forrest说自己的观点的，因为这样可以帮Forrest将逻辑串起来，或者指出一些错误。
