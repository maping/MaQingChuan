作业分析：
*Article (stopping public transportation）
这是Forrest第一次写“discuss both views and give your opinion”类型的作文。整体完成得不错。在段落结构和表达上需要注意～
1. Forrest记错这类作文的结构了。应该先分析不同意的一方，再分析同意的，这样整个思路比较顺畅（不同意 -> 同意 -> 结尾同意：XVV结构）。
上次课我们讨论之后，Forrest选了“不同意”，但是写作业的时候，可能有新的想法，又写了“同意”。但就算立场怎么变，记住支持的立场写后面就好了～
2. Forrest很喜欢用省略，把句子写得很简洁，比如“some believe everyone deserves…”，或许Forrest觉得看起来高级；但是，这种省略反而显得太生硬了。一般新闻报道会写得很简洁，因为有字数限制，特别是title。但是Forrest要分析自己的观点，所以要详细。如果Forrest能滔滔不绝写到超字数，那就更好了！！
3 用词不够准确，特别是常见的固定搭配（collocation），比如：overcome difficultities，而不是overcome challenges（中文也说“克服困难，迎接挑战”呀，逻辑相通的～）。课上分析了在这篇作文里，为什么用difficulties会比challenge更贴切。
3. 表扬Forrest，终于记得“on the one hand”和“one the other hand”是用于表达两个相反的观点的！
*Text C - Journal Entry
整体来说，Forrest对Journal的风格把握得不太好。但是对于bullet point的问题回答，是很不错的！
1. 在journal的开头，就写得很像正式的report。在不同的写作类型之间切换，对Forrest来说还是有难度的！之前我们写article比较多，现在换到比较轻松的风格，还有点不习惯，用了很多长难句，听起来很绕。
2. 有些表达很像中文直译，老毛病啦！还是要多多练习～
3. 部分句子写得不够详细，比如发生争吵的原因。Forrest可以假设阅卷老师没有读过原文，当看到这篇journal的时候，就可以知道事情的经过和结果。
4. “argument”和“separately”这两个词有可能是Forrest之前一直拼错并且不知道的！一定要注意啦！

课堂作文练习：
Text C - The business venture
1. 课上我们先分析了问题，然后带着问题分析文章。Forrest对文章提供的细节和线索不够敏感呢！比如文章提到的工蜂/蜜蜂，基本就是贯穿全文的。它不仅是两人看到的真实画面，更是一种象征。课上Forrest还反问老师，是怎么看出这些象征的哈哈！！
2. 这篇文章的故事本身很简单，就是两个朋友创业卖奶昔。但是，真正的难点是，理解隐藏主题，并且理解复杂的人物心理，还有人物性格。所以每次问Forrest，这篇文章讲了什么，他都能说出事情的经过，这是很好的plot retelling能力；但是要真正做到reflection，还是要多分析之前讲过的文章。Text C的每篇文章都很好！要好好利用哟～
