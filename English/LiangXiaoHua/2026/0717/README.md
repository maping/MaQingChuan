作业分析：
*Article
整体来说，结构完整，语法错误少，论点清晰，但是小问题还是重复出现：
1. People’s Opinions：因为Forrest在这里阐述两个相反的观点，最好用表示转折的逻辑词，我们之前经常用到的while。
2. Analysis/Example：感觉Forrest有点free style了。BP1的example在论点后，BP2则在分析后。课上我们分析了为什么Forrest的例子“不够例子”，并且讲解了怎样的逻辑是比较严谨的。
Forrest说学习了很长时间后需要放松，比如去散步，打篮球等等。但是Forrest可以加入“放松”的必要性：学习导致身心俱疲，所以才需要放松，而出门活动可以怎么改善…….
有很多内容可以写的～
3. Conclusion：有点长，且啰嗦，因为Forrest在重复之前两个BP段的内容。其实Forrest需要做的是总结+重申立场。
4. 用词：Forrest几乎能从每次作业里面学到新的表达，以及更正之前以为正确的搭配。比如英文里不说“take a walk”而会用“go for a walk”或者“take a stroll” 希望Forrest能整理复习。
5. Chingish: 还是会出现，但是比起上次，是有进步的，只有个别地方有问题。不过在写论点的时候看到chinglish的话就比较tricky了！就从考试来说，改卷老师都是很快浏览一遍文章的，重点也肯定是放在每段句首。所以再次叮嘱Forrest得检查，特别是每段句首。
6. 拼写错误：time、neighborhood两个单词拼错。再次强调要检查好再交作业
*Personal Statement
1. Story Bank是辅助Forrest收集和整理自己的故事的，所以最好要详细。比如rugby训练，有很多事情可以写，rugby只是一个大主题。Story Bank的最大作用就是看到experience，Forrest知道这段经历是从什么方向讲述的。
2. Forrest可能对moment有点误解啦；这是一个触动Forrest的瞬间，就像上次范文里面，作者的故事主线是去做志愿者，而开头的scene，描述的就是作者看到低保家庭住房环境的瞬间，文章描述了周围环境和作者内心的想法
3. Forrest大部分内容都描写得很好，有环境描写，也有心理描写，而且也会通过短句来强调。

PS：Part 2-4讲解
1. 课上我们根据Forrest的rugby故事，继续拓展剩下三个部分的内容。Forrest提出的很多点都是值得写的，不过他可能没意识到它们值得写。比如Forrest提到的“球商”，就是比起perseverance, resilience, discipline等更加特别和吸引人的角度。
2. Forrest切忌只是假大空地讲自己变得更加坚韧不拔了，一定要通过具体动作表现出来。
