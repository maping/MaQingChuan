作业分析：
*Summary
这次的Summary作业只有一个需要改的地方！好棒呀～
虽然Summary没有Article的正式度高，但是还是希望Forrest可以养成不写缩写的习惯（Don’t -> Do not等等），因为分开写在Summary和Article都是对的。

*Article
这次的Article作业整体思路清晰、语言表达较自然，结构完整；但在语法、句法、和行文逻辑上仍有一些可以改进的地方：
1. 标题。这段时间的标题在大小写、基础的语法上经常有小毛病，课上我们复习了作文标题的规则，希望Forrest多复习。
2. 每段的论点是这个段落的总起，所以后面的内容，都是围绕论点展开的。有很多细节要注意，比如分析部分的内容，最好按照论点提出的顺序展开论述，否则会比较乱。另外，论点提到的内容，下文一定要有对应的分析；论点没有提及的内容，下文就不要写啦！
3. 文章第一次用缩写，比如AI，记得要标注AI的全称；比如：AI (Artificial Intelligence)；
4. Body Paragraph 2的内容没有Body Paragraph 1的详尽。在之前的作业中，也出现类似的问题。Forrest可以对比一下两个段落的篇幅。这次body paragraph 2的内容展开层次单一，缺乏对原因和方式的解释。
5. 每个body段最后建议加一个简短的、扣题的结尾句，这样结构更完整。
 
 课堂作文练习：
今天的作文和之前写过的一次有点类似，想考察一下Forrest的知识迁移能力，因为它们本质的思维结构和论证逻辑是相通的。
1. 文章题目没有提到education，但是下面供Forrest参考的评论提到了；这种情况下，Forrest还是保险起见，以题目的主角“stricter punishment”作为讨论的出发点。
2. Forrest的想法真的很不错，他想到了stricter punishment会给人们造成精神上的影响。只要Forrest能有条理并且有说服力地进行阐述，就是好的点子。
3. 我们在分析论点的时候，Forrest经常会输出很多内容，包括论点之后的分析。在Forrest的一顿输出后，我们还是重新一步步来的原因是，希望Forrest分清论点和解释，因为他们考察了Forrest的总结和拓展的能力，而且它们在结构上，各有特色。
