# 围棋十诀
‌- 不得贪胜‌：局势领先后不应再为赢得更多而把局势搞复杂。

- 入界宜缓‌：打入对方阵势要徐徐图之，不求一击而得逞。
‌- 攻彼顾我‌：进攻对方时要注意保持节奏，有急有缓。
‌- 弃子争先‌：不要因小处纠缠而丧失先手主动。
‌- 舍小就大‌：每一步棋要选价值最大的那步走。
‌- 逢危须弃‌：形势不利时越逃可能死得越多，放弃越早损失越小。
‌- 慎勿轻速‌：对局时不要轻易落子，考虑成熟了再走。
‌- 动须相应‌：下棋要有全局观念，局部行动要为整体服务。
‌- 彼强自保‌：对方势强时赶快求活或走畅孤棋，不要以弱击强。
‌- 势孤取和‌：己方势力薄弱时避免硬拼，争取减少损失。‌‌‌
